# 1.0.1

## Bug Fixes

- Fixed `post_gen_project` hooks in `pypackage` and `pyservice` to use `cookiecutter._repo_dir` instead
of `cookiecutter._template`. When running cookiecutter against a GitHub URL (e.g. `gh:NERC-CEH/...`), `_template`
resolves to the URL rather than the local clone path, causing the shared hooks lookup to fail.
