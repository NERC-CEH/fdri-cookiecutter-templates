# 1.0.0

First stable release of the FDRI Cookiecutter templates.

## Features

- Python package template (`pypackage`) with pre-configured tooling: `ruff`, `pyright`, `pytest`, `coverage`, and `uv`
- Python service template (`pyservice`) for service-oriented projects
- Shared hooks across templates to reduce duplication
- Citation File Format (`.cff`) support for research software metadata
- Dependabot configuration with UV support for automated dependency updates
- Branch protection rules and GitHub Actions workflows included by default
- MIT licence as the default, with configurable copyright owner

## Changes since 0.1.0

- Split into separate `pypackage` and `pyservice` templates
- Moved shared logic to `_shared/hooks` for reuse across templates
- Switched default licence to MIT and updated copyright to organisation
- Improved unit test performance
- Updated docs to reflect new repository name and added `mkdocs` for the root project
